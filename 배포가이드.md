## 🔌 Google Sheets API 연결 (선택사항)

데이터를 저장하려면 Google Sheets API가 필요합니다.

### 방법 1: 기존 API 사용
```javascript
// config.js 5번째 줄
API_URL: 'https://script.google.com/macros/s/AKfycbxFmlFLv1efE8jTMDxDQqpx7vN0hOi4gVAVa0krl1i7p0EO7S_-imOE7ntBuxJefv34HA/exec',
```

### 방법 2: 새 Google Sheets 생성
1. **Google Sheets** 새 문서 생성
2. 시트 이름: `consultations`
3. 첫 행 헤더 추가:
   ```
   id | customerName | phone | eventDate | eventTime | eventCategory | eventType | guestCount | mealType | selectedOptions | partners | promotionAmount | depositAmount | basePrice | mealPrice | optionPrice | totalPrice | balancePrice | notes | status | createdAt
   ```
4. **도구 → Apps Script**
5. 코드 붙여넣기 (별도 제공)
6. **배포 → 새 배포 → 웹 앱**
   - 실행: 나
   - 액세스: 모든 사용자
7. URL 복사 → config.js에 붙여넣기

## ⚠️ 문제 해결

### 문제 1: 페이지가 안 열려요
**원인**: GitHub Pages 배포 중  
**해결**: 2~3분 후 다시 시도

### 문제 2: "API를 설정하세요" 경고
**원인**: config.js의 API_URL이 비어있음  
**해결**: 
- UI만 테스트: 무시하고 진행
- 데이터 저장 필요: API_URL 설정

### 문제 3: 옵션이 안 보여요
**원인**: 행사 유형을 선택하지 않음  
**해결**: 
1. 카테고리 선택 (웨딩/행사)
2. 유형 선택 (마이크로웨딩/돌잔치 등)
3. 옵션 자동 표시

### 문제 4: 협력업체 입력란이 안 보여요
**원인**: 체크박스를 체크하지 않음  
**해결**: 
1. 협력업체 체크박스 클릭
2. 입력란 자동 표시
