# 폴라그란데 예약관리 시스템

## 📋 프로젝트 개요
폴라그란데 웨딩/행사 예약 통합 관리 시스템

## ✨ 주요 기능
- 웨딩/행사 상담 등록 및 관리
- 실시간 캘린더 뷰
- 자동 가격 계산
- 선택 옵션 및 협력업체 관리
- A4 세로 2페이지 인쇄
- 모바일/태블릿/PC 반응형

## 🚀 설치 가이드

### 1단계: Google Sheets 설정
1. 새 Google 스프레드시트 생성
2. 시트 이름: "예약관리"
3. 첫 행에 헤더 입력:
```
ID | 고객명 | 연락처 | 행사일자 | 행사시간 | 예상인원 | 행사카테고리 | 행사유형 | 
음식종류 | 1인식대 | 기본금액 | 추가식대 | 옵션금액 | 프로모션 | 프로모션금액 | 
총금액 | 계약금 | 잔금 | 지급방식 | 계약상태 | 특이사항 | 옵션정보 | 협력업체정보 | 등록일
```

### 2단계: Google Apps Script 배포
1. 스프레드시트 → 확장 프로그램 → Apps Script
2. google-apps-script.js 코드 붙여넣기
3. 배포 → 새 배포
4. 유형: 웹 앱
5. 실행 계정: 나
6. 액세스 권한: 모든 사용자
7. 배포 URL 복사

### 3단계: config.js 수정
```javascript
const CONFIG = {
    API_URL: '여기에_배포된_URL_입력',
    ...
}
```

### 4단계: 서버 배포
- GitHub Pages 또는
- Netlify 또는
- 로컬 서버
