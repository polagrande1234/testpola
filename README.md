# 폴라그란데 예약관리 시스템

## 🎯 주요 기능

### ✅ 완료된 기능
- ✨ 웨딩 카테고리 유형 변경 (마이크로웨딩, 하우스웨딩, 가든웨딩)
- ✨ 웨딩 식대 계산 (24인까지 포함, 25인부터 추가)
- ✨ 행사 카테고리 유형 변경 (돌잔치, 가족행사, 기업행사, 대관)
- ✨ 행사 식대 계산 (양식/한식 선택, 기업행사/대관 직접입력)
- ✨ 선택옵션 체크박스 (7가지 + 기타옵션 직접입력)
- ✨ 협력업체 체크박스 (7가지, 금액 제외, 내용만 입력)
- ✨ A4 2페이지 인쇄 디자인 (연베이지톤)
- ✨ 모바일/태블릿/PC 반응형 UI

## 📋 설치 가이드

### 1단계: Google Sheets 생성
1. 새 구글 스프레드시트 생성
2. 시트 이름을 "예약관리"로 변경
3. 첫 행에 다음 헤더 입력:
   ```
   ID | 고객명 | 연락처 | 행사일자 | 행사시간 | 예상인원 | 행사카테고리 | 행사유형 | 
   기본금액 | 추가식대 | 옵션금액 | 프로모션 | 프로모션금액 | 총금액 | 계약금 | 
   잔금 | 지급방식 | 계약상태 | 특이사항 | 등록일
   ```

### 2단계: Google Apps Script 설정
1. 스프레드시트에서 `확장 프로그램` > `Apps Script` 클릭
2. `google-apps-script.js` 파일의 코드를 복사하여 붙여넣기
3. `배포` > `새 배포` 클릭
4. 유형: `웹 앱`
5. 실행 계정: `나`
6. 액세스 권한: `모든 사용자`
7. 배포 후 나오는 URL 복사

### 3단계: config.js 수정
1. `config.js` 파일 열기
2. `API_URL` 값을 2단계에서 복사한 URL로 변경
3. 저장

### 4단계: GitHub Pages 배포
1. GitHub 저장소 생성
2. 모든 파일 업로드
3. Settings > Pages > Source를 `main branch`로 설정
4. 생성된 URL로 접속

## 💰 가격표

### 웨딩 (24인 식대 포함)
- 마이크로웨딩: 4,200,000원
- 하우스웨딩: 4,900,000원
- 가든웨딩: 6,200,000원
- 25인부터 1인당 55,000원 추가

### 행사
- 돌잔치: 2,000,000원
- 가족행사: 2,500,000원
- 기업행사: 직접입력
- 대관: 직접입력

### 식사 가격
- 양식: 55,000원/인
- 한식: 59,000원/인

### 선택 옵션
1. 공간대여 패키지: 600,000원
2. 폴라돌상 패키지: 500,000원
3. 그란데돌상 패키지: 700,000원
4. 생신상 패키지: 200,000원
5. 음향·마이크: 80,000원
6. 애프터파티 패키지: 1,500,000원
7. 기타옵션: 직접입력

### 협력업체 (금액 계산 제외)
- 사회자, 촬영, 드레스, 메이크업, 생화장식, 부케, 기타
